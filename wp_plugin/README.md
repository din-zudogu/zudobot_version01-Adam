# Zudobot WordPress Plugin (PATH 3)

Deploy to a WordPress site:

1. Zip the `zudobot-embed/` folder (must contain `zudobot-embed.php`).
2. WordPress Admin → Plugins → Add New → Upload Plugin.
3. Activate **Zudobot AI Chat Integration**.
4. Open **Zudobot Chat** menu → Connect via OAuth.

Source of truth: `integrations/path3-marketplace/wordpress-plugin/zudobot-embed.php`
