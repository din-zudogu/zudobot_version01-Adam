🤖 Zudobot Widget Test Page

หน้านี้ใช้สำหรับทดสอบ widget ในเครื่อง

📋 Setup

  1. รัน node scripts/seed-test-tenant.mjs เพื่อรับ API key
  2. แทน YOUR_PUBLIC_KEY ด้านล่างด้วย key ที่ได้
  3. รัน API: cd apps/api && npm run dev
  4. เปิดไฟล์นี้ในเบราว์เซอร์

💬 Widget จะปรากฏที่มุมขวาล่าง

คลิกไอคอน 💬 เพื่อเริ่มคุยกับ น้องซูโด้

data-api-key="YOUR_PUBLIC_KEY"
data-api-url="http://localhost:4000"
